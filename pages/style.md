An ongoing curation of my style choices[^1], in order of increasing controversy:

[^1]: H/t to <a href="https://farcaster.xyz/kepano" target="_blank">@kepano</a> for this <a href="https://stephango.com/style" target="_blank">personal style collection concept</a>.

| Topic | Rule |
| :--- | :--- |
| Oxford comma | `yes` |
| Em dash | `yes`[^2] |
| Username | `@berensp`[^3] |
| TP orientation | `over` |
| Necktie knot | `Windsor` |
| Coffee order | `black`[^4] |
| Beer order[^5] | `amber lager` |
| Cocktail order | `Manhattan` |
| Book format | `physical` |
| Dates | `YYYY-MM-DD`[^6] |
| Times | `hh:mm` |

[^4]: Black is for good coffee and my home brew. If it's time for something fancier then "un cortado" (or "Gibraltar" as it were in San Francisco).

[^5]: I was going to add an entry for "Wine order" with "Côtes du Rhône" (or some other varietal with *l'accent circonflexe)*, but two alcoholic entries is enough—plus I'm a [low-key oenophile](/wine.html).

[^6]: <a href="https://xkcd.com/1179/" target="_blank">ISO 8601</a>—and other international standards where possible.
