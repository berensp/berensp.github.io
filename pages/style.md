A collection of my style choices[^1] in order of increasing controversy:

[^1]: H/t to <a href="https://stephango.com/style" target="_blank">Stephen Ango</a> for this style page concept.

| Style | Choice |
| :--- | :--- |
| Dates | `yyyy-mm-dd`[^2] |
| Times | `HH:MM` |
| oxford comma | `enable` |
| em dash | `yes but slow down`[^3] |
| voice | `conversational`, `wry`, `(parenthetical)` |
| TP orientation | `over`, you psychopath |

[^2]: <a href="https://xkcd.com/1179/" target="_blank">ISO 8601</a>

[^3]: AI has kind of ruined this as [Christy Bieber explains](https://read.technically.dev/p/ai-and-the-em-dash).
